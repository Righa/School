## Getting started

1. Installing dependencies

- install python

- install python pip

- install flask

- other dependencies

```
pip install flask-login
```
```
pip install bcrypt
```
```
pip install flask-wtf
```
```
pip install flask-sqlalchemy
```
```
pip install wtforms-sqlalchemy
```
```
pip install tensorflow
```
```
pip install numpy
```

2. Running

- open terminal
- cd to the School directory

```
python app.py
```
- or if using python3

```
python3 app.py
```
- open your browser

open http://127.0.0.1:5000/
